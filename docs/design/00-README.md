# Obrato — Design

**Obrato** — AI-ассистент прораба (web-PWA). Из замеров комнат + состава работ
детерминированно считает количества материалов по нормам, собирает локализованный
список закупок по этапам стройки и редактируемые 2D-чертежи. Инструмент прораба —
не дизайнера и не конечного заказчика.

> Рабочее название: **Obrato** (от порт. *obra* «стройка» + *-ato* «возведено/сделано»;
> читается на PT/EN/RU). Выбрано 2026-05-29 после проверки ~12 кандидатов на занятость —
> единственное без коллизий в нише. Отвергнуты как занятые: Pronto, Obra, Prumo (BR),
> Konstru, Buildora, Prazio, Levato.
> ⚠️ Подтвердить вручную: домен (obrato.com/.app/.pt), имя в Google Play Console,
> TM через INPI Portugal / EUIPO (класс 9 + 42).
