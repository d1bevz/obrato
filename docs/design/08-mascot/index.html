AMIGO

Amigo — спокойный, компетентный напарник-прораб: каска (единственный цветной акцент) и фирменные усы Дали, всегда закрученные вверх. Он зарабатывает место, делая работу — сообщает состояние, смягчает честную неопределённость, празднует результат. Минимализм iOS + запоминаемость Telegram. Не блокирует быстрый путь, уважает reduced-motion.

SVG + CSS · без Lottie/JS-либ тех: гл. 09 ① · презентация 0 новых сущностей данных эстетика: гл. 08
