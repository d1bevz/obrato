Персонаж Obrato: интерактивная анимация

Состояния персонажа привязаны к действиям интерфейса. Он помогает понять следующий шаг, подтверждает сохранение и спокойно объясняет неопределённость, но не превращает рабочий экран в игру.

Интерактивное правило

  • Персонаж реагирует на ближайший support-контекст: next step, поле, canvas, offline, широкий диапазон, факт.
  • Тап по персонажу раскрывает короткое объяснение карточки; второй тап сворачивает.
  • Если пользователь активно вводит данные, idle-анимация приглушается и не спорит с фокусом.

Технический контракт

  • MVP: SVG/CSS внутри SupportCard, intent приходит из состояния экрана.
  • Brand pass: Rive/Lottie state machine с тем же API и SVG fallback.
  • Без звуков, без layout shift, с поддержкой prefers-reduced-motion.
