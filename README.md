# Obrato

**AI-ассистент прораба.** Мобильное приложение (Android), которое из замеров
помещения и состава работ детерминированно строит технические чертежи (план полов,
развёртки плитки, электрика), считает количества материалов по нормам расхода и
собирает локализованный список закупок (Leroy Merlin Portugal и локальные магазины
Лиссабона) с ценами, наличием и разбивкой по этапам стройки.

## Зачем

Прорабы теряют деньги из-за нехватки нужных материалов в нужный момент: нет
материала → простой → срыв следующего заказа (≈1000 €/день потерь). Корень — нет
времени и горизонта на планирование закупок. Obrato снимает планирование с головы
прораба и переводит замеры в точную смету и план закупок вовремя.

Это инструмент **прораба**, а не дизайнера и не конечного заказчика. Эстетику/дизайн
Obrato не генерирует — он считает, что и когда покупать.
